# Hello git

所以，把页面的http链接都换成https也是不对的。要想让https小灰锁变成安全的小绿锁其实非常简单，只需要在页面head里加入以下标签即可：

```
<meta http-equiv="Content-Security-Policy" content="upgrade-insecure-requests" />
```
